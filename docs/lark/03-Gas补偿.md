# AA Wallet Gas补偿机制

## 1. 补偿计算流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        输入参数                                  │
│                                                                 │
│   ┌─────────────┐  ┌─────────────┐  ┌─────────────┐            │
│   │   gasUsed   │  │  gasPrice   │  │ BNB/USDT    │            │
│   │ (交易消耗)  │  │ (当前gas价) │  │ (Oracle汇率)│            │
│   └─────────────┘  └─────────────┘  └─────────────┘            │
│                                                                 │
│   ┌─────────────┐                                              │
│   │   feeRate   │                                              │
│   │ (手续费率)  │  ← 10000 = 100%, 100 = 1%                     │
│   └─────────────┘                                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        计算过程                                  │
│                                                                 │
│  步骤①: BNB成本                                                 │
│  ┌─────────────────────────────────────────────┐              │
│  │                                             │              │
│  │  bnbCost = gasUsed × gasPrice               │              │
│  │                                             │              │
│  │  例: 85000 × 3Gwei = 0.000255 BNB           │              │
│  │                                             │              │
│  └─────────────────────────────────────────────┘              │
│                              │                                  │
│                              ↓                                  │
│  步骤②: USDT补偿                                                │
│  ┌─────────────────────────────────────────────┐              │
│  │                                             │              │
│  │  compensation = bnbCost × bnbPrice ÷ 1e18   │              │
│  │                                             │              │
│  │  例: 0.000255 × 600 = 0.153 USDT            │              │
│  │                                             │              │
│  └─────────────────────────────────────────────┘              │
│                              │                                  │
│                              ↓                                  │
│  步骤③: 手续费                                                  │
│  ┌─────────────────────────────────────────────┐              │
│  │                                             │              │
│  │  fee = compensation × feeRate ÷ 10000       │              │
│  │                                             │              │
│  │  例: 0.153 × 100 ÷ 10000 = 0.00153 USDT     │              │
│  │                                             │              │
│  └─────────────────────────────────────────────┘              │
│                              │                                  │
│                              ↓                                  │
│  步骤④: 总扣除                                                  │
│  ┌─────────────────────────────────────────────┐              │
│  │                                             │              │
│  │  total = compensation + fee                 │              │
│  │                                             │              │
│  │  例: 0.153 + 0.00153 = 0.15453 USDT         │              │
│  │                                             │              │
│  └─────────────────────────────────────────────┘              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ↓
┌─────────────────────────────────────────────────────────────────┐
│                        USDT转账                                  │
│                                                                 │
│   ┌─────────────────────────────┐                              │
│   │  user → Relayer             │  ← USDT补偿 (0.153 USDT)     │
│   └─────────────────────────────┘                              │
│                                                                 │
│   ┌─────────────────────────────┐                              │
│   │  user → feeRecipient        │  ← 手续费 (0.00153 USDT)     │
│   └─────────────────────────────┘                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

---

## 2. 计算公式表

| 步骤 | 公式 | 说明 |
|------|------|------|
| BNB成本 | `gasUsed × gasPrice` | 实际消耗的BNB数量 |
| USDT补偿 | `BNB成本 × BNBPrice ÷ 1e18` | 转换为USDT金额 |
| 手续费 | `USDT补偿 × feeRate ÷ 10000` | 平台收取的手续费 |
| 总扣除 | `USDT补偿 + 手续费` | 用户总共支付的USDT |

---

## 3. 计算示例

假设参数：
- gasUsed = 85,000
- gasPrice = 3 Gwei = 3 × 10^9 wei
- BNB/USDT = 600 (即 600 × 10^18 wei)
- feeRate = 100 (即 1%)

```
步骤① 计算BNB成本
────────────────────────────────────────
bnbCost = gasUsed × gasPrice
        = 85,000 × 3 × 10^9
        = 255,000,000,000,000 wei
        = 0.000255 BNB


步骤② 计算USDT补偿
────────────────────────────────────────
compensation = bnbCost × bnbPrice ÷ 1e18
             = 0.000255 × 600 × 10^18 ÷ 10^18
             = 0.153 USDT


步骤③ 计算手续费
────────────────────────────────────────
fee = compensation × feeRate ÷ 10000
    = 0.153 × 100 ÷ 10000
    = 0.00153 USDT


步骤④ 总扣除
────────────────────────────────────────
total = compensation + fee
      = 0.153 + 0.00153
      = 0.15453 USDT
```

**结果：**

| 项目 | 金额 |
|------|------|
| 用户支付 | **0.15453 USDT** |
| Relayer收到 | **0.153 USDT** |
| 平台收到 | **0.00153 USDT** |

## 4. 合约中的实现

```
┌─────────────────────────────────────────────────────────────────┐
│                Paymaster.executeBatch() 实现                    │
│                                                                 │
│  // 1. 记录执行前gas                                            │
│  uint256 gasBefore = gasleft();                                 │
│                                                                 │
│  // 2. 执行用户操作                                             │
│  for (i = 0; i < calls.length; i++) {                           │
│      (success, ) = calls[i].to.call(calls[i].data);             │
│      require(success, "Call failed");                           │
│  }                                                              │
│                                                                 │
│  // 3. 计算gas消耗                                              │
│  uint256 gasUsed = gasBefore - gasleft();                       │
│                                                                 │
│  // 4. 计算BNB成本                                              │
│  uint256 bnbCost = gasUsed × tx.gasprice;                       │
│                                                                 │
│  // 5. 获取BNB/USDT汇率                                         │
│  uint256 bnbPrice = oracle.getBNBPriceInUSDT();                 │
│                                                                 │
│  // 6. 计算USDT补偿                                             │
│  uint256 compensation = (bnbCost × bnbPrice) / 1e18;            │
│                                                                 │
│  // 7. 计算手续费                                               │
│  uint256 fee = (compensation × feeRate) / 10000;                │
│                                                                 │
│  // 8. 执行USDT转账                                             │
│  usdt.transferFrom(user, relayer, compensation);                │
│  usdt.transferFrom(user, feeRecipient, fee);                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
