# AA Wallet 用户使用流程

## 1. 新用户完整流程

```
┌─────────────────────────────────────────────────────────────────┐
│                      新用户使用流程                              │
│                                                                 │
│  步骤①: 准备                                                    │
│  ┌───────────────────────────────────────────────────────┐     │
│  │                                                        │     │
│  │   用户需要有少量 BNB (用于7702授权)                     │     │
│  │   用户需要有 USDT 余额                                 │     │
│  │                                                        │     │
│  └───────────────────────────────────────────────────────┘     │
│                              │                                  │
│                              ↓                                  │
│  步骤②: 7702授权                                                │
│  ┌───────────────────────────────────────────────────────┐     │
│  │                                                        │     │
│  │   用户钱包签名 authorization                           │     │
│  │   Relayer 提交 setCode 交易                            │     │
│  │   用户 EOA 绑定 Simple7702Account 合约代码             │     │
│  │                                                        │     │
│  │   结果: 用户 EOA 现在拥有合约代码                       │     │
│  │                                                        │     │
│  └───────────────────────────────────────────────────────┘     │
│                              │                                  │
│                              ↓                                  │
│  步骤③: Gasless USDT转账                                        │
│  ┌───────────────────────────────────────────────────────┐     │
│  │                                                        │     │
│  │   用户输入: 目标地址 + 转账金额                         │     │
│  │   用户钱包签名 batch 内容                              │     │
│  │   Relayer 提交 executeBatch 交易                       │     │
│  │                                                        │     │
│  │   合约执行:                                            │     │
│  │   • 验证签名                                           │     │
│  │   • 执行 approve + transfer                            │     │
│  │   • 计算 Gas 补偿                                      │     │
│  │   • 从用户 USDT 扣除补偿                               │     │
│  │                                                        │     │
│  │   结果: 转账成功，用户无需持有 BNB                      │     │
│  │                                                        │     │
│  └───────────────────────────────────────────────────────┘     │
│                              │                                  │
│                              ↓                                  │
│  步骤④: 清除7702(可选)                                          │
│  ┌───────────────────────────────────────────────────────┐     │
│  │                                                        │     │
│  │   用户钱包签名清除 authorization                       │     │
│  │   Relayer 提交 setCode(空地址)                         │     │
│  │   用户 EOA 恢复为普通账户                              │     │
│  │                                                        │     │
│  │   结果: 用户恢复为普通 EOA，需要 BNB 支付 Gas          │     │
│  │                                                        │     │
│  └───────────────────────────────────────────────────────┘     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 3. USDT Gasless转账详细流程

```
用户(7702)            后端               Relayer           Paymaster合约
  │                    │                  │                    │
  │ 输入目标地址+金额  │                  │                    │
  │───────────────────→│                  │                    │
  │                    │                  │                    │
  │ 钱包签名batch      │                  │                    │
  │←───────────────────│                  │                    │
  │                    │                  │                    │
  │ 签名内容:          │                  │                    │
  │ ┌────────────────┐ │                  │                    │
  │ │ user: 用户地址 │ │                  │                    │
  │ │ calls: [       │ │                  │                    │
  │ │  approve(PM,amt)│ │                  │                  │
  │ │  transfer(to,amt)│ │                 │                  │
  │ │ ]              │ │                  │                    │
  │ │ nonce          │ │                  │                    │
  │ └────────────────┘ │                  │                    │
  │                    │                  │                    │
  │ 用户确认签名       │                  │                    │
  │───────────────────→│                  │                    │
  │                    │                  │                    │
  │                    │ 选择空闲Relayer  │                    │
  │                    │─────────────────→│                    │
  │                    │                  │                    │
  │                    │                  │ 构建交易           │
  │                    │                  │ executeBatch       │
  │                    │                  │───────────────────→│
  │                    │                  │                    │
  │                    │                  │                    │ ┌───────────┐
  │                    │                  │                    │ │验证层    │
  │                    │                  │                    │ │• Relayer │

  │                    │                  │                    │ ┌───────────┐

  │                    │                  │                    │ └───────────┘
  │                    │                  │                    │
  │                    │                  │←───────────────────│
  │                    │                  │ 交易完成           │
```

## 5. 用户7702状态生命周期

```
                              ┌─────────────┐
                              │   创建EOA   │
                              └─────────────┘
                                    │
                                    ↓
                              ┌─────────────┐
                              │    EOA      │ ← 纯EOA状态
                              │ (普通账户)  │   需要 BNB 支付 gas
                              └─────────────┘
                                    │
                                    │ 提交 7702 授权
                                    ↓
                              ┌─────────────┐
                              │ 授权Pending │
                              └─────────────┘
                                    │
                          ┌─────────┼─────────┐
                          ↓                   ↓
                    ┌──────────┐        ┌──────────┐
                    │ 授权成功 │        │ 授权失败 │
                    └──────────┘        └──────────┘
                          │                   │
                          ↓                   │
                    ┌──────────────────────────┐
                    │       Auth7702           │ ← 已绑定合约
                    │                          │   无需 BNB
                    │  • 可执行 batch 操作     │   USDT 支付 Gas
                    │  • 无需持有 BNB          │
                    │  • USDT 支付 Gas         │
                    │                          │
                    └──────────────────────────┘
                          │
                          │ 执行 Gasless 转账
                          ↓
                    ┌──────────────────────────┐
                    │     TransferReady        │ ← 转账状态
                    │                          │
                    │  转账成功 → 保持状态     │
                    │  转账失败 → Retry        │
                    │                          │
                    └──────────────────────────┘
                          │
                          │ 提交清除
                          ↓
                    ┌─────────────┐
                    │ 清除Pending │
                    └─────────────┘
                          │
                    ┌─────┼─────┐
                    ↓           ↓
              ┌────────┐  ┌──────────────────────────┐
              │清除成功│  │ 保持 Auth7702           │
              └────────┘  │                          │
                    │     │                          │
                    ↓     └──────────────────────────┘
              ┌─────────┐
              │ 回到EOA │ ← 恢复普通账户
              │         │   需要 BNB 支付 gas
              └─────────┘
```

## 6. 用户体验说明

```
┌─────────────────────────────────────────────────────────────────┐
│                     用户需知                                     │
│                                                                 │
│  7702授权前:                                                    │
│  • 用户需要有少量 BNB (用于 Relayer 提交授权交易)               │
│  • 用户需要有 USDT 余额                                         │
│                                                                 │
│  7702授权后:                                                    │
│  • 用户 EOA 拥有 Simple7702Account 的代码                       │
│  • 用户可以进行 Gasless USDT 转账                               │
│  • 每笔转账自动从 USDT 扣除 Gas 补偿                            │
│  • 用户无需持有 BNB                                             │
│                                                                 │
│  清除7702后:                                                    │
│  • 用户恢复为普通 EOA                                           │
│  • 需要自己有 BNB 支付 Gas 才能操作                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
